## 📋 Descripción

Tarjeta informativa para la materia de **Programación Web**, que muestra los datos del curso de manera visual y elegante. Incluye una animación de hover suave y un diseño centrado y responsivo.

## ✨ Características

- 📌 Diseño centrado con **Flexbox**
- 🎨 Fondo personalizado con tema Bootstrap (`bg-info`)
- 🖼️ Imagen de portada del curso
- 🏷️ Badges informativos (grupo, materia, aula)
- 🖱️ Efecto hover con elevación suave (`transform` + `box-shadow`)
- 📱 Layout responsivo gracias a Bootstrap 4
